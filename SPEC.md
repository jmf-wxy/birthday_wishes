# 生日庆祝网站 - SPEC.md

## 1. Concept & Vision

一个温馨、喜庆的生日庆祝网站，让朋友和家人可以为寿星送上祝福。网站风格充满欢乐和惊喜，包含动态蛋糕、吹蜡烛、烟花庆祝等互动元素，营造出温馨又热闹的生日派对氛围。整体感觉应该是：温暖、惊喜、有趣、让人想反复访问。

## 2. Design Language

### 美学方向
喜庆温馨风格，融合现代动画效果和传统生日元素，使用渐变色彩和柔和的动画营造温暖感。

### 色彩方案
- Primary: `#FF6B9D` (粉色 - 主色调，代表温馨浪漫)
- Secondary: `#FFD93D` (金黄色 - 代表喜庆和祝福)
- Accent: `#6BCB77` (绿色 - 蛋糕装饰)
- Background: `linear-gradient(135deg, #667eea 0%, #764ba2 50%, #f093fb 100%)` (紫色渐变背景)
- Text: `#FFFFFF` (白色文字)
- Cake: `#F8E1A6` (奶油色蛋糕)

### 字体
- 标题: "Ma Shan Zheng", cursive (中文手写风格)
- 正文: "Noto Sans SC", sans-serif
- Fallback: system-ui, sans-serif

### 空间系统
- 基础间距: 8px
- 卡片圆角: 16px
- 容器最大宽度: 1200px
- 组件间距: 24px

### 动效哲学
- 入场动画: 弹跳效果 (cubic-bezier(0.68, -0.55, 0.265, 1.55))
- 悬停效果: 轻微放大和阴影加深
- 庆祝动画: 烟花粒子效果、星星闪烁
- 蜡烛火焰: 持续摇曳动画

## 3. Layout & Structure

### 页面结构
1. **Hero Section**: 全屏展示，带有生日标题和蜡烛动画
2. **祝福墙**: 显示所有祝福语的卡片网格
3. **送祝福表单**: 输入祝福语的表单
4. **庆祝特效层**: 烟花和彩带动画覆盖层
5. **设置面板**: 设定寿星名字的弹窗
6. **分享面板**: 生成和复制分享链接的弹窗

### 响应式策略
- Mobile (< 768px): 单列布局，祝福卡片堆叠
- Tablet (768px - 1024px): 双列布局
- Desktop (> 1024px): 三列布局，大尺寸动画

## 4. Features & Interactions

### 核心功能
1. **吹蜡烛互动**
   - 点击蛋糕区域激活吹蜡烛动画
   - 蜡烛火焰摇曳后熄灭
   - 播放吹气音效
   - 熄灭后触发烟花庆祝

2. **祝福墙**
   - 显示祝福语卡片，包含祝福者姓名和内容
   - 祝福语使用 localStorage 持久化存储
   - 祝福按时间倒序排列
   - 祝福卡片有随机颜色主题

3. **送祝福表单**
   - 输入姓名（必填）
   - 输入祝福语（必填，最多200字）
   - 提交后卡片动画添加到祝福墙
   - 表单验证和错误提示

4. **庆祝动画**
   - 烟花粒子效果
   - 彩色纸屑飘落
   - 点击任意位置触发小烟花

5. **寿星名字设置**
   - 点击右上角⚙️按钮打开设置弹窗
   - 输入寿星名字并保存
   - 页面标题和主标题会自动更新为"🎂 [名字]生日快乐 🎂"
   - 名字保存在 localStorage 中持久化
   - 保存成功后触发烟花庆祝

6. **分享功能**
   - 点击左上角"🔗 分享"按钮打开分享弹窗
   - 自动生成包含寿星名字的分享链接
   - 链接格式: `?name=寿星名字`
   - 一键复制链接功能
   - 朋友打开链接后自动显示对应寿星的名字
   - 祝福数据通过 URL 参数共享（需要服务器端存储支持，当前版本仅支持本地 localStorage）

### 边界情况
- 空祝福墙: 显示引导性占位文字
- 表单空提交: 显示错误提示
- 长祝福语: 自动截断显示，悬停显示完整内容

## 5. Component Inventory

### 蛋糕组件
- 3层蛋糕造型
- 蜡烛（可点击吹灭）
- 樱桃装饰
- States: 点燃状态、吹灭状态

### 祝福卡片
- 圆角卡片
- 顶部彩色装饰条
- 祝福者姓名（加粗）
- 祝福语内容
- 时间戳
- States: 默认、悬停（放大阴影）

### 表单组件
- 输入框: 圆角边框，聚焦时高亮
- 提交按钮: 渐变背景，悬停效果
- States: 默认、聚焦、错误、成功

### 烟花粒子
- 圆形小粒子
- 多种颜色随机
- 从中心向外扩散动画

### 设置按钮
- 右上角固定位置
- 圆形按钮，齿轮图标
- 悬停时旋转30度

### 分享按钮
- 左上角固定位置
- 圆角矩形，渐变绿色背景
- 悬停时上浮效果

### 弹窗组件
- 毛玻璃背景遮罩
- 居中白色卡片
- 圆角24px
- 点击遮罩可关闭
- 动画: 缩放+淡入效果

### 设置弹窗
- 输入框: 寿星名字输入
- 保存按钮: 渐变粉色背景

### 分享弹窗
- 分享链接输入框(只读)
- 复制按钮: 紫色背景
- 复制成功后按钮文字变为"已复制!"

## 6. Technical Approach

### 技术栈
- 纯 HTML5 + CSS3 + JavaScript
- 无框架依赖
- CSS 动画和 transitions
- Canvas API 用于烟花效果
- localStorage 用于数据持久化
