# Supabase 数据共享设置指南

## 步骤 1: 创建 Supabase 账号

1. 访问 https://supabase.com
2. 点击 "Start your project"
3. 使用 GitHub 账号登录（推荐）
4. 免费计划足够使用

## 步骤 2: 创建新项目

1. 登录后点击 "New Project"
2. 填写项目信息：
   - **Name**: `birthday-wishes` (或你喜欢的名字)
   - **Database Password**: 设置一个强密码（记住它！）
   - **Region**: 选择离你最近的区域（如 ` Southeast Asia` 东南亚 或 ` Northeast Asia` 东北亚）
3. 点击 "Create new project"
4. 等待 2 分钟项目创建完成

## 步骤 3: 获取 API 密钥

1. 项目创建完成后，进入项目 dashboard
2. 点击左侧菜单 **Settings** → **API**
3. 找到以下两个值：

### URL
```
https://xxxxx.supabase.co
```
这就是 `YOUR_SUPABASE_URL`

### anon public
```
eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
```
这就是 `YOUR_SUPABASE_ANON_KEY`

## 步骤 4: 创建数据表

1. 在左侧菜单点击 **SQL Editor**
2. 点击 **New query** 新建查询
3. 粘贴以下 SQL 代码：

```sql
-- 创建祝福表
create table blessings (
    id bigint generated by default as identity primary key,
    created_at timestamp with time zone default timezone('utc'::text, now()) not null,
    name text not null,
    message text not null,
    color text
);

-- 允许公开访问（无需登录即可读取和写入）
alter table blessings enable row level security;

-- 允许所有人读取和插入祝福
create policy "Allow all to select" on blessings for select using (true);
create policy "Allow all to insert" on blessings for insert with check (true);
```

4. 点击 **Run** 执行

## 步骤 5: 更新网站代码

在 `index.html` 中找到以下两行（约在第 1472-1473 行）：

```javascript
const supabaseUrl = 'YOUR_SUPABASE_URL';
const supabaseKey = 'YOUR_SUPABASE_ANON_KEY';
```

替换为你在步骤 3 获取的真实 URL 和密钥：

```javascript
const supabaseUrl = 'https://xxxxx.supabase.co';
const supabaseKey = 'eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...';
```

## 步骤 6: 部署更新后的网站

将修改后的 `index.html` 重新部署到 GitHub Pages 或其他托管服务。

---

## 完成！

现在祝福墙的数据会自动同步到 Supabase 数据库，朋友在不同设备上访问都能看到相同的祝福！

## 注意事项

- Supabase 免费计划每月有 500MB 数据库限制，一般够用
- 如果数据量很大，可以考虑清理旧祝福
- API 密钥已经设置为匿名访问，无需担心安全问题
